## Platform-Specific Shortcuts

### Windows/Linux
- Use `Ctrl` for all shortcuts
- Use `Alt` for alternative shortcuts

### macOS
- Use `Cmd` instead of `Ctrl`
- Use `Option` instead of `Alt`
- Use `Delete` instead of `Backspace`

## Customizing Shortcuts

You can customize keyboard shortcuts:

1. Open Settings (`Ctrl+,`)
2. Go to "Keyboard Shortcuts"
3. Click on shortcut to edit
4. Press new key combination
5. Click "Save"

**Tips:**
- Avoid conflicts with browser shortcuts
- Use memorable combinations
- Export/import custom shortcuts
- Reset to defaults anytime

## Learning Tips

**Start with Basics:**
- Master general shortcuts first
- Learn navigation shortcuts
- Practice editing shortcuts

**Build Muscle Memory:**
- Use shortcuts consistently
- Avoid using mouse for common actions
- Practice daily

**Create Cheat Sheet:**
- Print this reference
- Keep it visible
- Highlight your favorites

**Gradual Learning:**
- Learn 2-3 new shortcuts per week
- Focus on your workflow
- Don't memorize all at once

## Shortcut Conflicts

If shortcuts don't work:

1. Check browser shortcuts
2. Check OS shortcuts
3. Check extension shortcuts
4. Customize conflicting shortcuts

**Common Conflicts:**
- `Ctrl+W` - Browser close tab
- `Ctrl+T` - Browser new tab
- `Ctrl+N` - Browser new window
- `F5` - Browser refresh

**Solutions:**
- Use alternative shortcuts
- Disable browser shortcuts
- Use fullscreen mode
- Customize shortcuts

## Productivity Tips

**Workflow Efficiency:**
- Use `Ctrl+D` to duplicate nodes quickly
- Use `Ctrl+G` to group related nodes
- Use `F5` to run without clicking
- Use `Ctrl+S` to save frequently

**Navigation Speed:**
- Use `Space+Drag` instead of scrollbars
- Use `Ctrl+1` to see full workflow
- Use `Home/End` to jump to start/end
- Use arrow keys for precise positioning

**Editing Speed:**
- Use `Enter` to edit nodes quickly
- Use `Tab` to navigate parameters
- Use `Ctrl+Enter` to save and continue
- Use `Esc` to cancel quickly
