## 💡 视觉层次

1. 页面标题
2. 待审核数量徽章
3. 批量操作按钮

### 次要元素 (中等显眼)

2. 助理名称和图标
3. 状态标签

4. 提交时间

## 🎨 设计原则

### 1. 清晰性
- 信息层次分明
- 重要操作突出显示
- 状态一目了然

### 2. 一致性
- 统一的颜色方案
- 一致的间距和对齐
- 标准的交互模式

### 3. 效率性
- 批量操作支持
- 快速搜索和筛选
- 一键审核

### 4. 友好性
- 清晰的提示信息
- 确认对话框防误操作
- 及时的反馈

## 📐 尺寸规范

### 间距
- 页面边距: 24px
- 卡片内边距: 24px
- 元素间距: 12px
- 按钮间距: 8px

### 字体
- 页面标题: 20px, 600
- 助理名称: 14px, 600
- 正文: 14px, 400
- 辅助文字: 12px, 400

### 组件尺寸
- 头像: 40px
- 按钮高度: 32px
- 输入框高度: 32px
- 表格行高: 64px
