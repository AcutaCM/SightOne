## 🐛 常见问题

### 问题 1: 页面报错
**检查**: 
- 浏览器控制台是否有错误
- HeroUI 是否正确安装
- sonner 是否正确安装

**解决**:
```bash
npm install @heroui/react sonner --legacy-peer-deps
```

### 问题 2: Toast 不显示
**检查**: 
- layout.tsx 中是否添加了 Sonner Toaster
- 是否有多个 Toaster 冲突

**解决**: 确认 layout.tsx 包含:
```tsx
import { Toaster as SonnerToaster } from "sonner";
// ...
<SonnerToaster position="top-right" richColors />
```

### 问题 3: 样式不正确
**检查**:
- Tailwind CSS 是否正确配置
- HeroUI 主题是否正确加载

**解决**: 刷新页面或重启开发服务器
