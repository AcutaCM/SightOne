# 市场 Home 页面测试指南

## 快速测试步骤

### 1. 启动应用

```bash
cd drone-analyzer-nextjs
npm run dev
```

### 2. 访问市场 Home 页面

1. 登录应用
2. 打开市场（Marketplace）
3. 确认默认显示 Home 标签页

## 常见问题排查

### 问题 1：卡片悬停效果不工作
- **可能原因**：CSS transitions 未生效
- **解决方案**：
  - 检查浏览器控制台是否有错误
  - 确认 transition 属性正确设置
  - 尝试刷新页面

### 问题 2：响应式布局不正确
- **可能原因**：Grid 断点配置问题
- **解决方案**：
  - 检查 Ant Design Grid 的 gutter 设置
  - 确认 Col 组件的 xs/sm/md/lg/xl 属性
  - 调整浏览器窗口大小测试

### 问题 3：主题切换后颜色不对
- **可能原因**：CSS 变量未正确应用
- **解决方案**：
  - 检查 HeroUI 主题变量是否定义
  - 确认 globals.css 正确加载
  - 刷新页面重新加载主题

### 问题 4：动画卡顿
- **可能原因**：性能问题或浏览器限制
- **解决方案**：
  - 关闭其他标签页释放资源
  - 检查 CPU 使用率
  - 尝试其他浏览器

### 问题 5：移动端布局错乱
- **可能原因**：视口设置或 Grid 配置
- **解决方案**：
  - 检查 viewport meta 标签
  - 确认移动端断点（xs）配置
  - 使用浏览器的设备模拟器测试
