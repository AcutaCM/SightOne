## Important Notes

### What Happens When You Remove an Assistant?

1. **Removed from Your List**: The assistant is removed from your personal collection
2. **Still Available in Market**: You can add it back anytime from the market
3. **Chat History Preserved**: Your chat history with that assistant is kept
4. **Active Assistant Switch**: If you remove your currently active assistant, you'll be switched to the default "Just Chat" assistant

### Cannot Remove?

If you don't see a delete button:
- The assistant might be a system preset (cannot be removed)
- You might not have permission to remove it
- Check if you're logged in

### Undo Removal

To restore a removed assistant:
1. Open the Market (click "Browse Market" in empty state)
2. Find the assistant you removed
3. Click "Add to My Assistants" button

## Keyboard Shortcuts

Currently, there are no keyboard shortcuts for removing assistants. Use the mouse to click the delete button.

## Troubleshooting

### Delete Button Not Visible
- Make sure you're hovering over the assistant card
- Check if the assistant is in your user collection (not just in market)
- Verify you have permission to remove assistants

### Confirmation Dialog Not Appearing
- Make sure you clicked the delete button (not the card itself)
- Check if popups are blocked in your browser
- Try refreshing the page

### Error Message After Confirmation
- Check your internet connection
- Verify you're logged in
- Try refreshing the page and removing again
- Contact support if error persists

### Assistant Still Appears After Removal
- Refresh the page
- Check if the removal was successful (look for success message)
- Clear browser cache if issue persists

## Related Features

- **Add Assistant**: Add assistants from the market to your collection
- **Empty State**: When you remove all assistants, you'll see an empty state with a "Browse Market" button
- **Search Assistants**: Use the search bar to find assistants in your collection
