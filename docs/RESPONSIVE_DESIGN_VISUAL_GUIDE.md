## 触摸交互流程

### 点击反馈

```
1. 触摸开始 (touchstart)
   ┌─────────────┐
   │   卡片      │  ← 正常状态
   └─────────────┘

2. 触摸激活 (touchactive)
   ┌───────────┐
   │  卡片     │    ← 缩小到 96%，透明度 80%
   └───────────┘

3. 触摸结束 (touchend)
   ┌─────────────┐
   │   卡片      │  ← 弹回正常状态（弹簧动画）
   └─────────────┘
   
4. 触发操作
   → 打开助理详情
   → 触觉反馈（震动 20ms）
```

### 长按反馈

```
1. 触摸开始
   ┌─────────────┐
   │   卡片      │
   └─────────────┘
   
2. 保持 500ms
   ┌─────────────┐
   │ ┌─────────┐ │  ← 显示边框高亮
   │ │  卡片   │ │
   │ └─────────┘ │
   └─────────────┘
   
3. 触发长按
   → 显示上下文菜单
   → 触觉反馈（震动 50ms）
```

### 滑动手势

```
1. 向左滑动
   ┌─────────────┐
   │   卡片      │ ←───
   └─────────────┘
   
   → 显示"收藏"选项
   
2. 向右滑动
   ┌─────────────┐
    ───→ │   卡片      │
   └─────────────┘
   
   → 显示"分享"选项
```
