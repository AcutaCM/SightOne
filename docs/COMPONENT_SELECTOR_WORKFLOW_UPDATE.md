# 组件选择器 - 工作流组件更新

## 🎉 更新完成

已成功将最新版的 Tello 工作流组件添加到组件选择器中。

## 📍 在组件选择器中的位置

### 组件信息

- **ID:** `tello-workflow-panel`
- **名称:** Tello工作流面板 (新版)
- **分类:** 控制 (Control)
- **描述:** 可视化工作流编辑器，支持内联参数编辑、节点折叠、实时验证和AI生成

### 图标

使用了组合图标，包含：
- ✅ 勾选圆圈（表示任务完成）
- ⚡ 闪电（表示工作流执行）

### 分类标签

- **颜色:** 蓝色 (Blue)
- **类别:** 控制
- **标签:** 控制

## 🚀 如何使用

### 方法 1：通过组件选择器添加

1. 点击主界面的"添加组件"按钮
2. 在组件选择器中找到"Tello工作流面板 (新版)"
3. 点击组件卡片即可添加

### 方法 2：通过分类筛选

1. 打开组件选择器
2. 点击"控制"分类标签
3. 找到"Tello工作流面板 (新版)"
4. 点击添加

### 方法 3：搜索添加

1. 打开组件选择器
2. 浏览"控制"类别的组件
3. 找到带有"新版"标记的工作流面板
4. 点击添加

## 🎓 快速开始教程

### 第一步：添加组件

1. 打开组件选择器
2. 选择"Tello工作流面板 (新版)"
3. 添加到布局

### 第二步：创建工作流

1. 从节点库拖拽节点到画布
2. 连接节点形成工作流
3. 配置节点参数

### 第三步：编辑参数

1. 点击节点上的参数值
2. 输入新值
3. 自动验证和保存

### 第四步：运行工作流

1. 点击"运行"按钮
2. 观察节点状态变化
3. 查看执行结果

### 第五步：保存工作流

1. 点击"保存"按钮
2. 输入工作流名称
3. 保存到本地
