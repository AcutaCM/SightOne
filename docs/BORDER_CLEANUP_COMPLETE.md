# 边框清理完成 - 极简界面优化

## 🎯 目标
清理页面中多余的边框（border），让界面更加清爽简洁。

## ✅ 清理策略

### 1. **全局边框优化 (globals.css)**

#### 卡片和面板边框
```css
/* 清理前 */
border: 1px solid hsl(var(--heroui-divider));

/* 清理后 - 浅色主题 */
border: none;

/* 清理后 - 深色主题 */
border: 1px solid rgba(59, 130, 246, 0.08); /* 极轻边框 */
```

#### 输入容器边框
```css
/* 清理前 */
border: 1px solid hsl(var(--heroui-divider));

/* 清理后 */
border: none; /* 浅色主题 */
border: 1px solid rgba(59, 130, 246, 0.05); /* 深色主题 */
```

### 2. **设计原则**

- **浅色主题**: 完全移除边框，使用背景色区分
- **深色主题**: 保留极轻边框（透明度 0.05-0.08）
- **交互元素**: 悬停时显示轻微边框提示

### 3. **边框透明度标准**

- **无边框**: `none`
- **极轻**: `0.05-0.08`
- **轻微**: `0.10-0.15`
- **适中**: `0.20` (仅用于特殊情况)
