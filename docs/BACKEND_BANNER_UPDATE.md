#### 特性

1. **彩色输出**
   - 横幅：青色 (Cyan)
   - 分隔线：黄色 (Yellow)
   - 启动消息：绿色 (Green)

2. **品牌展示**
   - ASCII艺术字显示"SIGHT ONE"
   - 中文名称：瞰析 ONE

## 🎨 颜色代码说明

使用ANSI转义序列实现彩色输出：

- `\033[96m` - 青色 (Bright Cyan)
- `\033[93m` - 黄色 (Bright Yellow)
- `\033[92m` - 绿色 (Bright Green)
- `\033[0m` - 重置颜色
