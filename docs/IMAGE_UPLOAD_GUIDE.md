# 📸 图片上传功能使用指南

## 快速开始

### 1️⃣ 访问页面
```
http://localhost:3000/demo-report
```

### 2️⃣ 上传图片

#### 原始图片（左侧）
- 点击左侧上传区域
- 选择植株照片
- 查看预览

#### 遮罩图片（右侧）
- 点击右侧上传区域
- 选择病害标注图
- 查看预览

### 3️⃣ 生成报告
- 填写植株ID
- 选择严重程度
- 点击"生成演示报告"

---

## 🎯 三种使用模式

### 模式1：完全自定义
```
✅ 上传原始图片
✅ 上传遮罩图片
→ 使用你的图片生成报告
```

### 模式2：部分自定义
```
✅ 上传原始图片
❌ 不上传遮罩图片
→ 原始图用你的，遮罩图用默认
```

### 模式3：使用默认
```
❌ 不上传任何图片
→ 使用SVG演示图
```

---

## 💡 图片建议

### 原始图片
- 📐 尺寸：800x600 或更高
- 📦 格式：JPG, PNG
- 💾 大小：< 5MB
- 📷 内容：清晰的植株照片

### 遮罩图片
- 📐 尺寸：与原始图相同
- 📦 格式：PNG（支持透明）
- 💾 大小：< 5MB
- 🎨 内容：病害区域标注

---

## 🔄 管理图片

### 清除图片
点击图片上方的"清除"按钮

### 重新上传
清除后再次点击上传区域

### 查看状态
报告预览卡片显示：
- ✅ 已自定义（两张都上传）
- ⚠️ 部分自定义（上传一张）
- 🎨 使用默认（都未上传）
