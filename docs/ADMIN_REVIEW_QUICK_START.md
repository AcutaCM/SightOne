# 管理员审核页面 - 快速启动

## 🚀 快速开始

### 1. 访问页面
```
http://localhost:3000/admin/review
```


## 📋 核心操作

### 单个审核（3 步）
1. 点击"查看"按钮
2. 查看助理详细信息
3. 点击"通过并上架"或"拒绝"

### 批量审核（3 步）
1. 勾选要审核的助理
2. 点击"批量通过"或"批量拒绝"
3. 确认操作

### 搜索筛选（2 步）
1. 在搜索框输入关键词
2. 选择状态筛选器

## 🎯 状态流转

```
用户提交
   ↓
[待审核] ──→ 管理员审核 ──→ [已发布] ✅ 上架商城
   ↓                          
   └──────────────────────→ [已拒绝] ❌ 不上架
```

## 📊 数据结构

```typescript
interface Assistant {
  id: string;              // 唯一标识
  title: string;           // 助理名称
  desc: string;            // 描述
  emoji: string;           // 图标
  prompt: string;          // 系统提示词
  tags?: string[];         // 标签
  isPublic: boolean;       // 是否公开
  status: 'pending';       // 状态：待审核
  author: string;          // 作者
  createdAt: Date;         // 创建时间
  reviewedAt?: Date;       // 审核时间
  publishedAt?: Date;      // 发布时间
}
```
