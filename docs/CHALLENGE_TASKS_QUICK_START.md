# 挑战卡任务快速入门

## 快速开始

### 1. 创建8字飞行任务

```typescript
// 在工作流编辑器中：
1. 从节点库拖拽"8字飞行"节点到画布
2. 双击节点配置参数：
   - 半径: 100cm
   - 速度: 50%
   - 循环: 2次
```

## 评分标准

### 8字飞行
- **轨迹精度** (70%): 与理想8字轨迹的偏差
- **完成时间** (30%): 与预期时间的比较
- **满分条件**: 偏差<10%, 时间在预期±20%内

### 穿越障碍
- **完成率** (80%): 成功穿越的障碍物比例
- **时间效率** (20%): 完成时间与超时时间的比率
- **满分条件**: 全部穿越, 时间<50%超时时间

### 精准降落
- **降落精度** (70%): 最终位置误差
- **尝试次数** (30%): 调整次数越少越好
- **满分条件**: 误差≤精度要求, 1次成功

## 常见问题

### Q: 如何提高8字飞行评分？
A: 
- 选择合适的半径（100-150cm较稳定）
- 降低速度提高精度（30-50%）
- 确保环境空旷，减少干扰

### Q: 穿越障碍失败怎么办？
A:
- 增加安全边距（20-30cm）
- 降低飞行速度（30-40%）
- 检查障碍物位置是否合理

### Q: 精准降落误差大怎么办？
A:
- 增加最大尝试次数（3-5次）
- 放宽精度要求（15-20cm）
- 确保降落区域有明显标记

## 进阶技巧

### 1. 组合任务
```
起飞 → 8字飞行 → 悬停(2秒) → 穿越障碍 → 精准降落
```

### 2. 条件分支
```
8字飞行
  ↓
判断评分 > 80?
  ├─ 是 → 穿越障碍（困难模式）
  └─ 否 → 穿越障碍（简单模式）
```

### 3. 循环训练
```
开始
  ↓
循环 3次:
  ├─ 8字飞行
  ├─ 记录评分
  └─ 悬停(5秒)
  ↓
生成训练报告
  ↓
结束
```

## 安全提示

⚠️ **重要安全事项**:
1. 确保飞行区域空旷，无障碍物
2. 保持安全距离（至少3米）
3. 准备好紧急停止按钮
4. 检查电池电量（建议>50%）
5. 在室内飞行时注意高度限制
