## Accessibility

All permission states include:
- Visual indicators (icons, colors)
- Text labels (clear messages)
- Tooltips (hover explanations)
- ARIA labels (screen readers)
- Keyboard navigation (tab order)

## Responsive Behavior

Permissions work consistently across:
- Desktop (full buttons and labels)
- Tablet (compact buttons)
- Mobile (icon-only buttons with tooltips)

All permission checks remain the same regardless of device.
